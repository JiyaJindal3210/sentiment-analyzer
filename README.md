# 🚀 Sentiment Analyzer

## 📌 Overview

A machine learning-based web application that analyzes user reviews and classifies them as **Positive, Negative, .
It also provides insights through bulk review analysis and visualization.

---

## ✨ Features

* 🔍 Real-time sentiment prediction for user input
* 📂 Bulk review analysis using CSV/TXT upload
* 📊 Visualization of sentiment distribution
* 🧠 NLP-based preprocessing (tokenization, stopword removal, TF-IDF)
* ⚡ Fast and efficient ML models for classification

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Machine Learning:** Scikit-learn
* **NLP:** TF-IDF Vectorization
* **Libraries:** Pandas, NumPy, Matplotlib

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/sentiment-analyzer.git
cd sentiment-analyzer
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the application

```
streamlit run app.py
```

---

## 📊 How It Works

* User enters a review or uploads a file
* Text is preprocessed using NLP techniques
* TF-IDF converts text into numerical features
* Trained ML model predicts sentiment
* Results are displayed with visual insights
